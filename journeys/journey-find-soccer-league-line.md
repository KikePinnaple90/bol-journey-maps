# Player Journey Map — Find a Specific Soccer League Line

**Product:** BetOnline Sportsbook  
**Journey:** A punter tries to find pre-match, live, and outright markets for a specific soccer league (e.g. Premier League, La Liga) in a single session  
**Last updated:** 2026-04-15

---

## Stages

| Stage | Question | Description |
|---|---|---|
| Awareness | "How does a punter become aware of and navigate to a specific league's betting content?" | User arrives at BetOnline having already decided they want to bet on a specific soccer league. |
| Engage | "How do we capture interest and move them toward their target league?" | User orients on the sports homepage and looks for a path into the left navigation to find their league. |

## Steps by Stage

### Awareness → Engage
**Step 1: Land on Sports Home**  
Persona: Recreational Punter — Touchpoint: Sports Homepage  
User arrives knowing what they want. They scan the homepage to find a path to their target soccer league.

### Engage
**Step 2: Open Left Nav and Navigate to Soccer**  
Persona: Recreational Punter — Touchpoint: Left Navigation, Sport tree  
User expands the Soccer tree and scrolls looking for their target league as a single, clear entry.

**Step 3: Select the League from the Soccer Tree**  
Persona: Recreational Punter, Sharp — Touchpoint: Left Nav, Soccer league list  
User clicks their league. Currently routes to either Pre-match OR Live — not a unified hub. Outrights are entirely separate.

### Convert
**Step 4: Browse Pre-match Games and Markets**  
Persona: Recreational Punter, Sharp — Touchpoint: Pre-match Sportsbook, League view  
User browses upcoming games and markets. May click into individual games to evaluate before selecting.

**Step 5: Select Live Betting and Browse Live Games and Markets**  
Persona: Recreational Punter, Sharp — Touchpoint: Live Betting section, Soccer league filter  
User navigates to the separate Live Betting section and re-finds their soccer league by scrolling or filtering across all sports.

**Step 6: Go Back to Sports and Find Futures in Left Nav**  
Persona: Sharp, Recreational Punter (seasonal) — Touchpoint: Sports Left Nav, Futures section  
User must leave the sportsbook, navigate back, and locate Futures in the left nav — a completely separate context with no link from the league they were just browsing.

**Step 7: Find Specific Soccer League in Outrights**  
Persona: Sharp, Recreational Punter (seasonal) — Touchpoint: Futures Left Nav, Soccer league list  
User navigates inside Futures to find their specific soccer league. Structure and naming differ from main sportsbook.

**Step 8: Browse League Outrights and Markets**  
Persona: Sharp, Recreational Punter (seasonal) — Touchpoint: Outrights / Futures, League market view  
User browses futures markets (league winner, top scorer, relegation) and evaluates odds before selecting.

**Step 9: Select Bet and Add to Betslip**  
Persona: Recreational Punter, Sharp — Touchpoint: Game detail / Market view → Betslip  
User clicks an odds button to add a selection. May build a single or parlay. Places the bet.

### Excite
Bet placed successfully. Correct selection and stake confirmed in under 30 seconds from betslip open.

## Pain Points

| Stage | Step | Pain Points |
|---|---|---|
| Engage | Land on Sports Home | Homepage is promotion-heavy; direct league navigation not immediately visible. No "recently viewed leagues" shortcut. Live and Pre-match are separate top-level sections, creating early confusion. |
| Engage | Open Left Nav | League list is long with no live/pre-match/outright availability signals. No unified entry per league. Sharp bettors find navigation too slow. |
| Engage | Select League | No unified league hub — Pre-match and Live route to separate views. Outrights are in a completely separate section. Users must navigate 2–3 places to see full league picture. Multiple left nav entries for the same league. |
| Convert | Browse Pre-match | Not all markets accessible from a single entry. Real games duplicated in the list. Left nav game counter is inaccurate. Games going live disappear from pre-match without transition. |
| Convert | Browse Live | No direct path from pre-match league page to live markets for the same league. Live section shows all leagues alphabetically. No league context carried from pre-match session. Odds latency causes bet rejection. |
| Convert | Find Futures | Futures completely disconnected from league page. No cross-sell from pre-match or live. Soccer outrights not inside the Soccer sport tree. Naming conventions are inconsistent. |
| Convert | Browse Outrights | Same as above plus: journey to get here is too long; market availability is inconsistent across leagues. |
| Convert | Betslip | Betslip slow to update on odds change. Parlay across pre-match + live not supported. Mobile stake entry UX is poor. |

## Opportunities

| Stage | Step | Opportunities |
|---|---|---|
| Engage | Land on Sports Home | Surface direct league access on sports home without requiring left nav expansion |
| Engage | Open Left Nav | Show live/pre-match/outright availability badges next to each league name. Create a League Hub with all content accessible from a single nav entry. |
| Engage | Select League | Build a unified League Hub page with tabs for Pre-match / Live / Outrights at one URL. Deep-link league from anywhere to the hub. |
| Convert | Browse Pre-match | Merge duplicated games into a single entry. Unify left nav entries for the same league. Add SGP and Odds Booster CTAs on the league view. |
| Convert | Browse Live | Add a direct "Live Now" link on the league hub. Persist league filter when switching from pre-match to live. Order leagues by popularity within a sport. |
| Convert | Find Futures | Surface outright teasers on the league hub with a Futures tab. Add cross-sell strip in game detail views. Consolidate soccer outrights inside the Soccer sport tree. |
| Convert | Browse Outrights | Futures tab on unified league hub. Remove separate outright entry points, consolidate into league hub. Standardize outright type naming conventions. |
| Convert | Betslip | Improve live betting delay. Push users to update accept-odds preferences. Enable quick-stake buttons ($10 / $25 / $50 / $100). |
